S.M.A.R.T. = Secure Monitoring & Analysis of Real-time Traffic

Live Traffic Analysis using Machine Learning

SmartLog is a real-time, containerized log analysis system hosted on an Azure VM. It uses Filebeat, Kafka, and a Python ML pipeline with custom transformers to detect suspicious HTTP traffic. Logs are stored in PostgreSQL, while live dashboards are built in Grafana. Docker Compose manages all services, enabling modular deployment and easy scaling.
